* **9.11** 九一一事件，基地组织恐怖分子劫持四架飞机分别撞向世贸大厦、五角大楼、国会大厦（未成功）。计造成2977人死亡或失踪

* **9.1**-9.3 别斯兰人质事件。车臣武装分子在北奥塞梯共和国别斯兰市一所中学，劫持1200人为人质。最终造成333名人质死亡

### 2008
